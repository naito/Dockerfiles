biosim1-20-01
=============

Build
-----

* Build the Dockerfile to make a Docker image file "biosim1-20-01".
```bash
$ docker build -t biosim1-20-01 .
```

Usage
-----

* Run Docker image file "biosim1-20-01".
```bash
$ docker run -it -p 8888:8888 biosim1-20-01
```
